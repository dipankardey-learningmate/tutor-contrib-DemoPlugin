DemoPlugin plugin for `Tutor <https://docs.tutor.overhang.io>`__
===================================================================================

Installation
------------

::

    pip install git+https://github.com/dipankardey-learningmate/tutor-contrib-DemoPlugin.git

Usage
-----

::

    tutor plugins enable DemoPlugin
